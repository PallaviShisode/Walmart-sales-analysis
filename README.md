# Walmart-sales-analysis (SQL + Python Project)

📊 Project Overview

This project is an end-to-end data analysis solution designed to extract meaningful business insights from Walmart sales data.

It demonstrates:

Data cleaning and preprocessing using Python
Exploratory data analysis (EDA)
SQL-based business problem solving
End-to-end workflow from raw data to insights



🛠️ Tools & Technologies Used
Python (Pandas, NumPy)
SQL (MySQL / PostgreSQL)
Jupyter Notebook
VS Code



🔄 Project Workflow
1. Data Collection
Dataset obtained from Kaggle
2. Data Cleaning (Python)
Removed duplicates
Handled missing values
Fixed data types
3. Feature Engineering
Created new column: Total Amount
4. Data Analysis
Performed EDA using Pandas
Identified trends and patterns
5. SQL Analysis
Solved business problems using SQL queries


📈 Key Insights
Branch A generated the highest overall revenue
Food & Beverages category had the highest sales
Peak sales occurred during evening hours (6 PM – 9 PM)
E-wallet was the most preferred payment method
Certain cities showed consistently higher customer ratings


⚙️ How to Run the Project

Install required libraries:

pip install pandas numpy sqlalchemy mysql-connector-python psycopg2
Run Jupyter Notebook:
jupyter notebook
Execute Python notebooks for:
Data cleaning
Data analysis
Run SQL queries in your database environment
