# 📊 Walmart Sales Analysis (SQL + Python Project)

---

## 📌 Project Overview

This project is an end-to-end data analysis solution designed to extract meaningful business insights from Walmart sales data.

It demonstrates:

* Data cleaning and preprocessing using Python
* Exploratory Data Analysis (EDA)
* SQL-based business problem solving
* End-to-end workflow from raw data to insights

---

## 🛠️ Tools & Technologies Used

* Python (Pandas,numpy)
* SQL (MySQL )
* Jupyter Notebook
* VS Code

---

## 🔄 Project Workflow

### 1. Data Collection

* Dataset obtained from Kaggle

### 2. Data Cleaning (Python)

* Removed duplicates
* Handled missing values
* Fixed data types

### 3. Feature Engineering

* Created new column: **Total Amount**

### 4. Data Analysis

* Performed EDA using Pandas
* Identified trends and patterns

### 5. SQL Analysis

* Solved business problems using SQL queries

---


 ## 📈 Key Insights

* E-wallet and cash are the most commonly used payment methods
* Sales are highest during evening hours, indicating peak customer activity
* Top-performing categories drive a major share of total revenue
* Customer ratings vary across branches and product categories
* Payment preferences differ by branch, showing regional behavior patterns
* Some branches show a decline in year-over-year revenue
